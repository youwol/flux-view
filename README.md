# Installation, Build, Test

To install the required dependencies:

```shell
yarn
```
---
To build for development:

```shell
yarn build:dev
```

To build for production:

```shell
yarn build:prod
```
---


```shell
yarn test
```

Coverage can be evaluated using:
```shell
yarn test-coverage
```
---

To generate code's documentation:

```shell
yarn doc
```
